# DoAn-J2EE
Web Bán Phụ Kiện Thú Cưng
Admin:
-Quản lý tài khoản
-Tạo,xóa tài khoản
Chủ cửa hàng:
-Quản lý sản phẩm: thêm, xóa sửa sản phẩm
-Quản lý hóa đơn: xem, xác nhận hoặc từ chối đơn hàng, hoàn tắt đơn hàng
Khách hàng:
-Quản lý tài khoản:xem thông tin, sửa thông tin tài khoản
-Đặt hàng: thêm sản phẩm vào giỏ hàng hoặc mua ngay, thanh toán
-Xem lịch sử đơn hàng
Database liên kết với XAMPP là MySQL
Xây dựng theo kiến trúc MVC + Service + Repository của Spring Boot và Spring Data JPA.
